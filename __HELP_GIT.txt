Шаг 1: Перейти в папку с проектом
cd D:\gerb2tiff

Шаг 2: Удалить старую историю Git
Remove-Item -Recurse -Force .git

Шаг 3: Инициализировать новый репозиторий
git init

Шаг 4: Добавить удалённый репозиторий
git remote add origin https://github.com/DimaFantasy/gerb2tiff

Шаг 5: Закоммитить файлы и отправить изменения
git add .
git commit -m "Начальная версия"
git push --force origin master 

master имя текушего branch
